# Go Auth Server

Authentication server in Go.

### Install
Installing EdgeX Auth Server is trivial [`go get`](https://golang.org/cmd/go/):
```bash
go get github.com/drasko/edgex-auth
cd edgex-auth/cmd
go buld -o edgex-auth
./edgex-auth
```

### Features
- JWT
- User account creation
- MongoDB persistance
- Login and logout
- Token revocation
